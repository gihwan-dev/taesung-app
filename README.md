# taesung-app
B.sori 부산 엽합 캡스톤 경진 대회에서 진행한 앱 개발 입니다.

## 프로젝트 소개
pwa로 개발되었으며, 태성환경연구소에서 개발 및 판매하고 있는 악취포집기의 데이터를 모니터링 하고 나아가 제어하기 위한 앱을 개발하였습니다.

## 기능
- 로그인
- 회원가입
- 악취 포집기 모니터링




## 기술 스택

- **프레임워크:** NestJS
- **데이터베이스:** (예: MongoDB, PostgreSQL 등)
- **인증:** (JWT, OAuth 등 사용된 인증 방식)
- **문서화 도구:** Swagger (예시)
- **테스트:** Jest (예시)
